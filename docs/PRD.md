## 🎯 핵심 정보

**목적**: 친목 모임 주최자가 카카오톡에서 겪는 참여자 관리, 정산, 공지 문제를 한 곳에서 해결하는 웹 서비스

**사용자**: 20~40대 직장인 및 동호회 주최자가 모바일로 접근하여 초대 링크 공유, 참석 현황 확인, 정산 관리를 하는 사용자

## ⚡ 기능 명세 (MVP)

### 1. 핵심 기능

| ID       | 기능명                 | 설명                                               | MVP 필수 이유           | 관련 페이지                          |
| -------- | ---------------------- | -------------------------------------------------- | ----------------------- | ------------------------------------ |
| **F001** | 이벤트 생성            | 제목, 날짜, 장소, 인원, 비용 입력하여 모임 생성    | 주최자의 핵심 액션      | 이벤트 생성 페이지                   |
| **F002** | 초대 링크 생성 및 공유 | 고유 코드 기반 초대 링크 생성, 카카오톡 공유 버튼  | 참여자 모집의 핵심      | 이벤트 관리 페이지                   |
| **F003** | 참석 의사 표시         | 참석/불참/미정 선택 (로그인 없이 게스트도 가능)    | 가장 큰 Pain Point 해결 | 초대 링크 페이지, 이벤트 상세 페이지 |
| **F004** | 참여 현황 대시보드     | 참석/불참/미정 인원 실시간 표시                    | 주최자의 의사결정 기반  | 이벤트 관리 페이지                   |
| **F005** | 더치페이 정산          | 1인당 금액 자동 계산, 미납자 표시, 카카오페이 링크 | 정산 복잡도 해결        | 이벤트 관리 페이지                   |
| **F006** | 카풀 매칭              | 운전자 등록, 탑승자 신청, 매칭 관리                | 교통 편의성             | 이벤트 관리 페이지                   |
| **F007** | 공지사항 작성          | 주최자가 공지 작성, 참여자에게 표시                | 공지 관리의 중앙화      | 이벤트 관리 페이지                   |

## 🛠️ 기술 스택

### 🎨 프론트엔드

- **Next.js 15.5.3** (App Router) - React 풀스택 프레임워크
- **React 19.1.0** - UI 라이브러리 (최신 동시성 기능)
- **TypeScript 5.6** - 타입 안전성

### 🎨 스타일링 & UI

- **TailwindCSS v4** - 유틸리티 CSS (설정파일 없는 새로운 엔진)
- **shadcn/ui** - 고품질 React 컴포넌트
- **Lucide React** - 아이콘 라이브러리

### 📝 폼 & 상태

- **React Hook Form 7.x** - 폼 상태 관리
- **Zod** - 스키마 검증
- **Zustand** - 경량 상태관리

### 🗄️ 백엔드 & 데이터베이스

- **Supabase** - BaaS (PostgreSQL, 인증, 실시간)
- **PostgreSQL** - 관계형 데이터베이스
- **NextAuth.js** - 카카오 OAuth 2.0 인증

### 🚀 배포

- **Vercel** - Next.js 15 최적화 배포 플랫폼
