# 📊 Moim 린 캔버스 (Lean Canvas)

> **Moim**: 친목 모임 주최자가 카카오톡에서 겪는 참여자 관리, 정산, 공지 문제를 한 곳에서 해결하는 모바일 퍼스트 웹 서비스

---

## 1️⃣ 문제 (Problem)

### 주최자의 Pain Point

- **참여자 관리 복잡성**: 카카오톡 채팅방에서 참석 여부를 일일이 확인하고 정리해야 함
- **정산 복잡도**: 참가비 정산 시 누가 얼마를 내야 하는지 수동으로 계산하고 관리
- **공지 분산**: 중요한 일정 변경, 장소 변경 정보가 채팅에 묻혀 전달 누락 발생
- **재참여 어려움**: 과거 모임 정보 검색 어려움, 반복되는 모임 재설정 필요

### 참여자의 Pain Point

- **의사표시 번거로움**: 텍스트로 "참석", "불참" 표시해야 함
- **정보 수집 어려움**: 언제 어디서 몇 명이 오는지 한눈에 파악 불가
- **카풀 매칭 불편**: 운전자/탑승자 조율을 채팅으로 수동 진행

---

## 2️⃣ 고객 세그먼트 (Customer Segments)

### Primary Target

- **20~40대 직장인** (수도권 및 주요 도시)
- **동호회 운영자** (등산, 스포츠, 취미 클럽 등)
- **모임 주최 경험 있는 사용자**

### Secondary Target

- **회사 팀 이벤트 운영자**
- **소모임 리더**

### Early Adopter

- 카카오톡 그룹채팅 피로도가 높은 주최자 (월 2회 이상 모임)

---

## 3️⃣ 고유 가치 제안 (Unique Value Proposition - UVP)

**"카카오톡의 혼란을 벗고, 모임은 한 곳에서 쉽게"**

### 핵심 가치

1. **참여자 관리 자동화** → 초대 링크 공유로 즉시 참석 현황 파악
2. **정산 자동 계산** → 1인당 금액 자동 계산, 카카오페이 링크 자동 생성
3. **정보 중앙화** → 모든 정보(날짜, 장소, 공지)를 한 곳에서 관리
4. **카풀 매칭** → 운전자와 탑승자를 자동 연결

### 고객이 얻는 이득

- **시간 절약**: 정산 및 참석 확인에 걸리는 시간 80% 단축
- **정산 오류 제거**: 자동 계산으로 "누가 얼마 냈나?" 분쟁 해결
- **모임 운영 효율화**: 주최자 업무 시간 50% 감소
- **참여자 편의**: 클릭 몇 번으로 참석 여부 표시 및 공지 확인

---

## 4️⃣ 해결책 (Solution)

### MVP (최소 기능 제품)

| 기능               | 설명                                             | 대상          |
| ------------------ | ------------------------------------------------ | ------------- |
| **이벤트 생성**    | 제목, 날짜, 장소, 참가비 입력 후 초대 링크 생성  | 주최자        |
| **초대 링크 공유** | 고유 코드 기반 초대 링크, 카카오톡 버튼으로 공유 | 주최자        |
| **참석 의사 표시** | 로그인/게스트 모두 [참석/불참/미정] 선택 가능    | 참여자        |
| **현황 대시보드**  | 실시간 참석/불참/미정 인원 현황 표시             | 주최자        |
| **더치페이 정산**  | 참가비 ÷ 참석 인원 = 1인당 금액 자동 계산        | 주최자        |
| **카풀 매칭**      | 운전자 등록 → 탑승자 신청 → 자동 배치            | 전체          |
| **공지사항**       | 주최자가 공지 작성, 참여자가 모임 내에서 조회    | 주최자/참여자 |
| **대시보드**       | 주최 중/참여 중인 모임 목록 한눈에 조회          | 사용자        |

### 기술 구성

- **Frontend**: Next.js 15 + React 19 + TailwindCSS (모바일 퍼스트)
- **Backend**: Supabase (PostgreSQL + Auth)
- **인증**: 카카오 OAuth (회원가입 장벽 최소화)
- **배포**: Vercel

## 5️⃣ 채널 (Channels)

### Acquisition (고객 획득)

1. **SNS/커뮤니티**
   - 인스타그램, 페이스북 직장인 커뮤니티
   - 네이버 카페 (동호회, 모임 관련)
   - 스포츠/취미 클럽 온라인 커뮤니티

2. **입소문 (Viral Loop)**
   - 초대 링크 공유 시 서비스 자동 노출
   - 카카오톡 공유 버튼 → 자동 마케팅

3. **제휴**
   - 카카오 T (카풀)와의 연동
   - 문화생활앱 (공연, 전시) 연동

### Activation (사용자 활성화)

1. **온보딩**: 첫 모임 생성까지 3단계 (< 2분)
2. **초대 링크**: 카카오톡 공유로 즉시 참여 가능
3. **푸시 알림** (Phase 4): 이벤트 D-1 리마인더

### Retention (사용자 유지)

1. **반복 사용 유도**: 다음 모임 자동 제안
2. **공지 알림**: 중요 업데이트 즉시 알림
3. **과거 모임 기록**: 지난 모임 추억 저장

### Revenue (수익화)

- MVP 단계: 무료 (사용자 확보 중심)
- v2 단계: 프리미엄 기능 (고급 정산, 통계)

---

## 6️⃣ 수익 흐름 (Revenue Streams)

### MVP 단계 (0개월~3개월)

- **무료 모델**: 사용자 확보 및 검증에 집중

### v1.5 단계 (3개월~6개월)

- **결제 수수료**: 정산 시 카카오페이 수수료 (0~3%)

### v2 단계 (6개월+)

1. **프리미엄 구독**
   - 월 $2.99: 고급 정산 기능 (자동 정산 리마인더, 통계)
   - 월 $4.99: 팀 관리 기능 (다중 관리자, 모임 그룹화)

2. **B2B 라이센스**
   - 회사 팀 이벤트 관리 시스템
   - 동호회 플랫폼 화이트라벨

3. **광고** (선택적)
   - 카테고리별 맞춤 광고 (레스토랑, 액티비티)

- **CAC (Customer Acquisition Cost)**: $0 (입소문/초대 기반)

## 9️⃣ 경쟁 우위 (Unfair Advantage)

### 기술적 우위

1. **초대 링크 기반 진입 장벽 제거**
   - 카카오톡 공유로 즉시 참여 가능 (회원가입 불필요)
   - 경쟁사 대비 진입 마찰 80% 감소

2. **카카오톡 생태계 시너지**
   - 카카오 OAuth로 기존 사용자 기반 활용
   - 카카오 T, 카카오페이와의 연동 가능

3. **모바일 퍼스트 UX**
   - 손가락 한 번으로 모든 작업 완료
   - 3단계 이내에 모임 생성/참여 가능

### 시장 우위

1. **선점 효과 (First Mover Advantage)**
   - 카카오톡 모임 관리 전용 솔루션 부재
   - 초기 사용자 확보 시 네트워크 효과 극대화

2. **바이러스 루프 (Viral Loop)**
   - 초대 링크 공유 = 자동 마케팅
   - CAC 거의 0에 가까움

3. **수집 데이터 (Data Moat)**
   - 모임 패턴, 참여 데이터 축적
   - 향후 추천 알고리즘, 통계 분석으로 차별화

## 🎯 주요 성공 가정 (Key Assumptions)

### 검증 필요 가정

1. ✅ **모임 주최자는 현재 카카오톡에서 참여자 관리에 피로감을 느낀다**
   - 테스트 방법: 타겟 사용자 인터뷰 (10명)

2. ❓ **초대 링크 기반 플랫폼이 카카오톡 대체 가능하다**
   - 테스트 방법: MVP 사용성 테스트 (실제 모임 5개)

3. ❓ **정산 자동화 기능이 구매 욕구를 이끈다**
   - 테스트 방법: 프리미엄 변환율 측정

4. ❓ **수도권 20~40대가 타겟 세그먼트로 충분하다**
   - 테스트 방법: 지역별 사용율 분석
